Namaz Vakitleri API

Diyanet Isleri Baskanligi verileriyle - Ucretsiz statik JSON API

Nasil Calisir? Bu API, GitHub Pages uzerinden statik JSON dosyalari olarak sunulmaktadir. Her ayin basinda GitHub Actions otomatik olarak Diyanet API'sinden verileri ceker ve gunceller.

Ulkeler

GET /data/countries.json

Eyaletler / Sehirler

GET /data/states/{countryId}.json

Ornek: /data/states/2.json (Turkiye)

Ilceler

GET /data/cities/{stateId}.json

Ornek: /data/cities/500.json (Adana ilceleri)

Namaz Vakitleri (Aylik)

GET /data/prayer-times/{cityId}/{yyyy-MM}.json

Ornek: /data/prayer-times/17885/2026-04.json (Devrekani - Nisan 2026)

Gunluk Icerik (Ayet, Hadis, Dua)

GET /data/daily-content/{yyyy-MM-dd}.json

Ornek: /data/daily-content/2026-04-10.json

Sehir Arama Indeksi

GET /data/city-index.json

Tum ilcelerin ID ve isimlerini icerir. Mobil uygulamada arama icin kullanilabilir.

Meta Bilgi

GET /data/meta.json

Son guncelleme tarihi, sonraki guncelleme, toplam kayit sayilari.
